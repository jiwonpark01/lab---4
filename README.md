# 프로젝트 제목
## 1. 개요
### 이 프로젝트는 props, state, hook, useEffect 를 이용한 Todo List 만드는 프로젝트입니다.

## 2. 주요 기능 설명
이 프로젝트의 주요 기능은 다음과 같습니다.
- 할 일 입력
- 할 일 추가
- 할 일 완료/미완료 전환
- 할 일 삭제
- 남은 할 일 개수 표시
- localStorage 저장
- localStorage 복원
